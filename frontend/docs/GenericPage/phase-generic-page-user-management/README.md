# Phase: Generic Page cho User Management

Mục tiêu: áp dụng kiến trúc Generic Page (config-driven + slot pattern) cho `UserManagementPage` để làm mẫu nhân rộng cho các entity khác. Tài liệu này liên kết nhanh tới các phần chi tiết:

- `architecture-notes.md`: Bối cảnh, mapping kiến trúc vào page hiện tại.
- `implementation-checklist.md`: Checklist áp dụng chung cho mọi page.
- `user-management-example.md`: Cấu hình cụ thể cho User Management (table, form, filters, actions).
- `open-questions.md`: Câu hỏi/giả định cần xác nhận.
- `research-summary.md`: Tổng hợp nguồn tham khảo (Context7, code_research).
